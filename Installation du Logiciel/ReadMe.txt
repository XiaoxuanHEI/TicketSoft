Le projet est un projet Java-Maven (vous devez avoir Maven installé)
Nous utilisons Java 8 pour ce projet (OracleJDK8)

Avant de lancer le serveur il faut installer le GoogleSDK pour se connecter à la base de donnée et demander à Robin (robin.malmasson@u-psud.fr) d'ajouter votre compte google parmi les comptes autorisés à accéder à la BD.

Il est préférable, pour éviter les incompatibilités, de lancer le client sur un navigateur moderne comme Firefox ou Edge.
